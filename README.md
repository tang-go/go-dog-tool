<h1 align="center">go-dog-tool</h1>
<div align="center">
go-dog对应使用的自定义插件
</div>

### 使用

- clone 仓库
```bash
git clone https://github.com/tang-go/go-dog-tool.git
cd go-dog-tool
```

- 启动服务注册中心
```
cd go-dog-find
go build
./go-dog-find
```

- 启动网管
```
cd go-dog-gw
go build
./go-dog-gw

浏览器输入 http://127.0.0.1/swagger/index.html 可访问文档
```

- 启动服务管理中心
```
cd go-dog-ctl
go build
./go-dog-ctl
```
